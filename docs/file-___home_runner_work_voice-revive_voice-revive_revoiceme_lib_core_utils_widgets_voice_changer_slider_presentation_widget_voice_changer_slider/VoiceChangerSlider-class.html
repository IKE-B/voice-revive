VoiceChangerSlider

VoiceChangerSlider class Widget

A widget for controlling a certain voice changing parameter via a user controllable slider.

The VoiceChangerSlider consists of a a descriptive title and a slider for increasing or decreasing the value. Optionally, the slider can be activated or deactivated by a switch. The switch's state is correlated with the _VoiceChangerSliderState._isActive variable.
