AudioChanger

AudioChanger class Widget

A widget for manipulating the audio I/O via a user controllable slider.

The AudioChanger consists of a slider, a switch, buttons for increasing or decreasing the value and a descriptive title. It consumes a value from the parent widget and reflects it on the slider. The value can be changed by the slider or the buttons. The parent widget is notified via the onChangedValue function. The maxValue controls the maximum logical value of the slider. The switch's state is correlated with the _AudioChangerState._isActive variable. It controls whether this widget interferes with the given value or not.
